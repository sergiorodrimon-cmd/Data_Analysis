# Análisis de Churn de Clientes

## Descripción del Proyecto
Este proyecto analiza el abandono de clientes (churn) para identificar patrones y variables
asociadas a la pérdida de usuarios.

## Objetivo de Negocio
Reducir la tasa de churn mediante la identificación temprana de segmentos de alto riesgo.

## Datos
Dataset de clientes que incluye:
- Información demográfica
- Uso del servicio
- Historial de facturación
- Estado de churn

## Proceso de Análisis
1. Limpieza y preparación de datos
2. Análisis exploratorio (EDA)
3. Comparación de clientes activos vs churn
4. Identificación de variables clave

## Análisis Claves
- Tasa de churn global
- Churn por segmento
- Variables más asociadas al abandono

## Herramientas Utilizadas
- Python
- pandas
- seaborn / matplotlib
- Jupyter Notebook

## Insights Principales
- Identificación de perfiles con mayor probabilidad de abandono
- Variables de comportamiento más relevantes

## Recomendaciones
- Acciones de retención focalizadas
- Intervenciones tempranas según perfil
